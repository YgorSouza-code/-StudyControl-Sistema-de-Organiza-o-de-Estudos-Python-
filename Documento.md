 StudyControl – Sistema de Organização de Estudos em Python

O StudyControl é um sistema desenvolvido em linguagem Python com o objetivo de auxiliar estudantes na organização de suas tarefas acadêmicas, permitindo um melhor controle de prazos e acompanhamento de progresso.

 Tecnologias Utilizadas
Python – Linguagem principal utilizada no desenvolvimento
SQLite – Banco de dados relacional leve e nativo, dispensando instalação externa

 Funcionalidades do Sistema:
O sistema possui as seguintes funcionalidades principais:
Cadastro de usuários
Autenticação (login)
Criação de tarefas com data de entrega
Listagem de tarefas cadastradas
Marcação de tarefas como concluídas
Visualização do progresso das atividades

 Funcionamento:
O sistema foi desenvolvido utilizando conceitos fundamentais da programação, tais como:
Estruturas condicionais (if/else)
Estruturas de repetição (while)
Funções para organização do código
Entrada de dados via teclado (input)
Saída de informações no terminal (print)
Além disso, o sistema realiza:
Entrada de dados: informações fornecidas pelo usuário
Processamento: cálculos de progresso e manipulação de tarefas
Saída de dados: exibição das informações organizadas no terminal

 Banco de Dados:
O sistema utiliza o SQLite, permitindo:
Armazenamento persistente de usuários e tarefas
Relacionamento entre usuários e suas atividades
Execução de comandos SQL (INSERT, SELECT, UPDATE)
Problema Resolvido

O StudyControl foi desenvolvido para solucionar um problema comum entre estudantes: a falta de organização nos estudos.
Com o sistema, é possível:
Controlar tarefas acadêmicas
Acompanhar prazos
Visualizar o progresso das atividades
Melhorar a produtividade

Reflexão final Como a sua solução pode gerar impacto positivo utilizando conhecimentos de computação? 
  A solução desenvolvida, o StudyControl, gera impacto positivo ao aplicar conhecimentos de computação para resolver um problema real: a dificuldade de organização nos estudos.
 Por meio da utilização da linguagem Python e do banco de dados SQLite, o sistema permite armazenar, gerenciar e recuperar informações de forma eficiente. Isso possibilita que o usuário tenha controle sobre suas tarefas, prazos e progresso, promovendo uma melhor gestão do tempo e aumento da produtividade.
 Além disso, o uso de conceitos fundamentais da computação, como estruturas condicionais, laços de repetição, funções e manipulação de dados, garante que o sistema funcione de maneira lógica, organizada e automatizada. A persistência de dados com o banco de dados permite que as informações não sejam perdidas, tornando o sistema confiável para uso contínuo.
 O impacto positivo também está na acessibilidade da solução, já que o sistema é simples, leve e não exige tecnologias complexas, podendo ser utilizado por qualquer estudante com acesso básico a um computador.
 Dessa forma, o StudyControl demonstra como a computação pode ser aplicada para criar ferramentas que facilitam o dia a dia, auxiliam na organização pessoal e contribuem diretamente para um melhor desempenho acadêmico.

 Conclusão:
O projeto demonstra a aplicação prática dos conhecimentos adquiridos no curso, integrando programação, banco de dados e lógica computacional para resolver um problema real de forma simples e eficiente.
